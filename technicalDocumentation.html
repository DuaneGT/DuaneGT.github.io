Client - Side and Server - Side development

Client-side and server side development are two distinct aspects of web development that involve different technologies,responsibilities and functionalities.

Client-side development:

This refers to the activities that takes place in the user's device (most cases, a web browser) and involve the user interface and user experience.

  • Technologies used:

    • HTML (HyperText Markup Language), CSS (Cascading Style Sheets), JavaScript and other front-end frameworks like Angular and Vue.js

  • Responsibilities:

    • Creating the actual elements on a webpage using HTML and styling it using CSS.
    • Implementing interactive and dynamic features by using JavaScript.
    • Managing the presentation parts and the user interactions

  • Advantages:

    • Faster loading of webpages
    • Reduces server load since certain operations are performed on the client's side.
    • Better user experience with dynamic and interactive elements.
Server Side Development

Server side development involves activites that occur on the web server, managing and processing data, handling requests, and performing various server-related operations

  • Technologies Used:

    • Server-side languages such as PHP, Python(Django),Ruby (Ruby on Rails),Java (Spring), and JavaScript (Node.js)

  • Responsibilities:

    • Handling data storage and retrieval from databases.
    • Executing business logic and application functionality.
    • Managing user authentication and authorization.
    • Generating dynamic content to be sent to the client.

    Advantages:

    • Enhanced Security as sensitive operations are performed in the server.
    • Centralized control over data and business logic.
    • Better scalability for handling a large number of users.

HTML Formatting

There are numerous ways to format an element in HTML. Those examples are:

  • strong - makes the element affected to be displayed on bold text.
  • u - makes the element that is affected become underlined.
  • i - makes the element affected to be italic.
  • em - emphasizes the text affected.
  • abbr - defines a abbreviation or an acronym
